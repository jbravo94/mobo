Backup Process Support

A key responsibility of operating IT services is to ensure backups are properly performed (beside testing regularly the restore process). Especially understaffed IT teams most likely suffer from information overload (mails, dashboards, mobile notifications, etc.) where critical information can simply be overlooked. mobo supports here as an independent gadget aiming for disrupting the regular cognitive overload to shift attention.

Installation

  1. Organize Waveshare ESP32-S3 1.75inch AMOLED Round Touch Display Development Board with Case

    Order hardware from Amazon

  2. Download and flash firmware via Arduino Studio (Will be available soon)

    Refer to Source Code for details

  3. Configure required parameters via serial at baudrate 115200 

    > AUTH BackupBuddyAdmin
> PUT wifi-ssid GL-MT6000
> PUT wifi-password *****
> PUT ntp-server-dn at.pool.ntp.org
> PUT aws-lamdba-dn ozi5ej.lambda-url.eu-central-1.on.aws
> PUT aws-region eu-central-1
> PUT aws-access-key AKIAVL
> PUT aws-secret-key *****
> QUIT
    Restart mobo by unplug and replug

  4. Setup AWS Infrastructure and instrument applications with reporting routine (Will be documented soon)
